Jeopardy Question Analysis

I analyzed a dataset of around 200,000 Jeopardy questions to see how people should actually study for the game. I looked at things like how often answers show up in the question text, how often words repeat across episodes, and which topics appear more in high-value clues.

The main takeaway: don't try to "read into" the wording, Jeopardy mostly rewards real knowledge, and the same topics come up way more often than you'd expect.

What I found

Answers appear in the question text only ~6% of the time, so looking for hidden clues doesn't really work
87% of complex words in questions had already appeared in older episodes,  studying past questions is genuinely useful
No single term showed a statistically significant bias toward high-value questions (at least in the sample tested)
